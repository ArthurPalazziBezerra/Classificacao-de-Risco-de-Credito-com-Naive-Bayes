# Classificacao-de-Risco-de-Credito-com-Naive-Bayes
Este projeto tem como objetivo construir um modelo de classificação para prever o risco de crédito de clientes, utilizando o algoritmo Naive Bayes. A análise é baseada em uma base de dados com múltiplas variáveis financeiras e comportamentais, permitindo identificar padrões que podem indicar se um cliente é considerado bom ou mau pagador.

O processo tem início com o carregamento da base de dados original em Python, seguida da preparação das variáveis. Como o conjunto possui diversas colunas categóricas, é necessário transformá-las em valores numéricos. Para isso, cada atributo é codificado com seu próprio objeto LabelEncoder, garantindo que a conversão siga sempre o mesmo padrão tanto no conjunto de treinamento quanto no conjunto de teste. Essa padronização é fundamental para evitar inconsistências na hora de validar o modelo e também para realizar previsões futuras.

Após o pré-processamento, a base é dividida entre treinamento e teste, de forma que o modelo seja avaliado com dados que ele nunca viu antes. Em seguida, o classificador Naive Bayes é ajustado com os dados de treinamento, aprendendo as relações probabilísticas entre os atributos e a variável alvo. Com o modelo treinado, realizamos as previsões sobre o conjunto de teste, gerando a matriz de confusão e os cálculos de acurácia e erro. O uso da biblioteca Yellowbrick permite visualizar de maneira clara a matriz de confusão, facilitando a interpretação do desempenho.

Depois de avaliar o modelo, é possível utilizá-lo para prever novos registros. Para isso, um segundo arquivo contendo informações de um cliente inédito é carregado e passa pelo mesmo processo de transformação aplicado aos dados originais. Como os mesmos codificadores são reutilizados, os valores são traduzidos de forma consistente, permitindo que o modelo gere uma previsão confiável sobre o risco de crédito desse novo cliente. De forma geral, o projeto demonstra como aplicar Naive Bayes em um problema real de classificação, passando por todas as etapas essenciais: pré-processamento, treinamento, avaliação e previsão com dados novos.
