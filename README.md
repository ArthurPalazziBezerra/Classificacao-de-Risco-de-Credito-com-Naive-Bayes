# Classificacao-de-Risco-de-Credito-com-Naive-Bayes
Este projeto tem como objetivo construir um modelo de classificação para prever o risco de crédito de clientes, utilizando o algoritmo Naive Bayes. A análise é baseada em uma base de dados com múltiplas variáveis financeiras e comportamentais, permitindo identificar padrões que podem indicar se um cliente é considerado bom ou mau pagador.
